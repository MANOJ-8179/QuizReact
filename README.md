# 🎯 React Quiz Application

This project is a simple **Quiz Application built using React.js**.  
The application allows users to answer multiple-choice questions and displays the final score after completing the quiz.

The goal of this project is to demonstrate **React component structure, state management, and user interaction** in a quiz format. Quiz apps like this are commonly used for learning and knowledge testing in web applications. :contentReference[oaicite:1]{index=1}

---

# 🚀 Features

- Multiple choice quiz questions
- Interactive user interface
- Next question navigation
- Score calculation
- Result display after completing quiz
- Simple and responsive design
